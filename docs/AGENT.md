# AGENT.md

`docs` 配下で大きめの改修作業用ドキュメントを作る場合は、`docs/change_tasks/AGENT.md` の運用ルールに従ってください。

## ルール

- 大きめの改修テーマは `docs/change_tasks/` 配下で管理すること
- 新規改修時は `YYYY-MM-DD_改修内容` フォルダを作成すること
- 各改修フォルダには必ず以下を置くこと
  - `AGENT.md`
  - `CHECKLIST_改修作業名.md`
- 作業が進んだら、チェックリストを都度更新すること

## docs 配下の用途

- `docs/` には `change_tasks/` のほか、テストプレイ用の `index.html` とビルド済みのゲーム実体が入っている
- テストプレイ用のビルドを行った際は、`docs/` 配下にビルド成果物を配置すること

## 参照先
