# 状態理想挙動入力シート

このシートはフェーズ2用です。

---

## 0. ゲーム概要

- 女性向けワンシチュエーションエロゲ。
- マフィア幹部（イケメン）を緊縛拘束し、快楽堕ちさせる。
- Live2Dによる繊細な表情・身体表現がウリ。

## 共通方針

### 全体で大事にしたいこと

- このゲームで反応させるキャラクターは「反抗心の高い誇り高いマフィア幹部」であり、それを快楽落ちさせる事が最大の楽しみとなる。
- 基本的には高反抗のままゲームが進行し、「嫌なはずなのに興奮が高まり、気持ちよくなってしまう困惑」を与えて屈服させるのがゲームの見どころ。
- 一回クリアした後は「媚薬」が手に入り、任意のタイミングで使えるようになる。
使用した瞬間抵抗値を大幅に下げ、以降抵抗があがりづらくなり、AccliやSurrendなどのトロトロルートに入りやすくなる。
- 高反抗/媚薬ルートの分岐をはっきり見せつつも、媚薬を使わなくても執拗にプレイすればトロトロルートに入れるようなゲームバランスにする事が重要

### 状態遷移のポイント

- `Resistance` と `Drive` で状態差をはっきり出す
- `Arousal` は「感じている量」として使い、状態差の主軸にしすぎない。

　　しかし他条件を満たしたうえで遷移するタイミングの制御要因としてArousalを使うのは重要。低Arousalで高興奮状態/高Arousalで低興奮状態に遷移すると不自然になるから。

- `EdgeTension` は Main 強弱の駆け引きの中心に置く。

　　キャラの状態によって、EdgeTension突入時の処理には２つの役割がある。
　　①「我慢したい！イキたくない！」という意思の演出

　　②興奮最終局面で「ずっとイッてるのにまだ気持ちいい〜（気持ちよさが限界突破してる感にEgdeTを使う）」というフロー感の演出

- `Defensive / Acclimating` と `Surrendered / BrokenDown` の差がひと目で分かるようにする

　　Defenは高ストレスで我慢、Accliは受け入れて高まる経過演出。Surrenは気持ちよくてアンアンよがる様子、Brokenは完全理性崩壊でもっとアヘアヘしたり白目剥いてたりする感じでLive2Dをつくりたい

### 絶対に避けたい挙動

- どの状態も「Arousal が高いか低いか」だけで差が決まってしまうこと
- `Drive` が低いのに終盤状態っぽく見えること。
- 後半、DriveやFatigueが高止まりして操作した実感が湧かなくなること
- `BrokenDown` が単に `Surrendered` の上位互換にしか見えないこと
- `EdgeTension` が `Arousal` のおまけになり、駆け引きがなくなること

### 気持ちよく感じる進行

- Guarded/Defensive で拒絶されてるのに、ゴリ押しするとOverridden となり「イヤなのに身体が反応してしまう！！」という屈服をさせられるところ。また、そこから焦らしプレイでFrustに堕として「イヤなはずなのに体が求めてしまう…」という困惑をさせる過程がめちゃくちゃ楽しい。
- 操作を工夫して反抗を下げることに成功すると、Acclimating/Surrendered に質感が変化してキャラがとろけていく過程が楽しい
- 「感じている」だけではイかず、「我慢するのに耐えきれない」ことで `EdgeTension` からオーガズムに至る快感の駆け引き
- 快楽受容と理性崩壊が別軸で進み、最終的に BrokenDown で両方が極まる流れ

### 状態差として特に強く出したい軸

- Resistance / 受容　…状態遷移の横軸
- Drive / 理性崩壊　…状態遷移の縦軸
- Arousal / 快感量　…状態遷移のタイミング指定要素
- Fatigue / 消耗感　…良い意味でも悪い意味でも「やり過ぎ感」の演出。最終局面のEnd処理突入用。
- EdgeTension / イキそう感　…Resi高いときは我慢感、高Driveだと快感限界突破感
- DriveBias / オス・メス傾向　…キャラをどの方向の快感で堕とすのか？という愉しさ。オス傾向なら安直な気持ちよさで圧倒する方向性、メス傾向はキャラの男性性を蕩けさせてメス堕ちさせる懐柔のたのしみ。

## 1. 内部パラメータ

| パラメータ | 説明 | 値域 |
| --- | --- | --- |
| `Arousal` | 快感の反応量 | `0.0` ～ `1.0` |
| `Edge Tension` | オーガズム欲 | `0.0` ～ `1.0` |
| `Resistance` | 抵抗感（高=拒絶、低=受容） | `0.0` ～ `1.0` |
| `Fatigue` | 消耗・疲弊 | `0.0` ～ `1.0` |
| `Drive` | 理性崩壊の総量 | `0.0` ～ `1.0` |
| `DriveBias` | 崩壊の質（`-`=トロトロ寄り、`+`=アヘ顔寄り） | `-1.0` ～ `1.0` |
| `NeedMotion` | 腰の反応・求める動き（内部蓄積） | `0.0` ～ `1.0` |
| `RejectMotion` | 拒絶や苛立ちを表現するランダム発火モーション。Resisに応じて発火率などが変化（内部蓄積） | `0.0` ～ `1.0` |
| `RejectHab` | 安定した刺激には慣れ、急な刺激がくるとRejectMotion発火率を上げるための関数（内部蓄積） | `0.0` ～ `1.0` |

## 2. 入力定義

### 2.1 MainIntensity（ピストン強度）

- スライダーで制御（`0.0` ～ `1.0`）。
- 手を離してもその値で停止・継続。
- `isActive` フラグで停止状態と区別。

### 2.2 SubA（チンコ刺激）

- クリック or バイブトグル（`ON/OFF`）。
- `Arousal` 増加 / `Resistance` 低下 / `DriveBias+` 方向 / `Drive` 増加の傾向をもつ（状況によって可変）

### 2.3 SubB（乳首刺激）

- クリック or バイブトグル（`ON/OFF`）。
- `Arousal` 増加 / `Resistance` 増加（デフォルト） / `DriveBias-` 方向 / `Drive` 増加の傾向をもつ（状況によって可変）
- `SubBResistanceChange` を負値に設定すれば状態ごとに低下に変更可能。

### 2.4 SubA + SubB 同時入力（実装仕様）

- SubA/B はそれぞれ独立して加算（単体ONと同時ONで特別処理は変わらない）。
- DriveBias のみ例外: 同時ON時は合算値の絶対値を**マイナス方向に強制** `(-|SubAGain + SubBGain|)`。

## 3. 入力帯（Band）判定

- 状態ごとに `TolLow` / `TolHigh` が変わる。

| 入力帯 | 条件 |
| --- | --- |
| `Stop` | `isActive == false` |
| `Below` | `MainIntensity < TolLow` |
| `Within` | `TolLow <= MainIntensity <= TolHigh` |
| `Above` | `MainIntensity > TolHigh` |

## 4. 状態定義と遷移

### 4.1 状態ごとの閾値（現状実装内容）

| 状態 | TolLow | TolHigh | 備考 |
| --- | --- | --- | --- |
| ① `Guarded` | 0.30 | 0.50 | 狭い、中間のみ許容 |
| ② `Defensive` | 0.20 | 0.40 | 刺激に敏感で嫌がる範囲が広い |
| ③ `Overridden` | 0.30 | 0.70 | わりとなんでも `Within` |
| ④ `FrustratedCraving` | 0.25 | 0.55 | 弱い刺激が `Within`（焦らし感） |
| ⑤ `Acclimating` | 0.30 | 0.70 | ③と同程度 |
| ⑥ `Surrendered` | 0.20 | 0.80 | ほぼ全域 `Within` |
| ⑦ `BrokenDown` | 0.15 | 0.85 | ほぼ全域 `Within` |

# ① Guarded

## 一言定義

- 緊張しながらもまだ自分を保っている初期警戒状態

## 心理

- 拒絶/受容: やや拒絶寄りだが、防衛に振り切る直前ではない
- 主体性: 高い
- 恥じらい/警戒/強がりなど: 警戒、緊張、まだ余裕を装っている感じ

## パラメータ感

- Arousal: 低め。ゼロではないが、快感として認めていないのであげづらい。
- Drive: かなり低い。Sub刺激をうまく使ってやらないとあがりにくい。
- Resistance: 中高
- Fatigue: かなり低い
- EdgeTension: 我慢できる。まだ余力があるので我慢しやすい。
- DriveBias: まだ中立。ここから染めていく真っ白なパレット

## Main入力の感じ方

- Stop: 少し落ち着ける。Guarded の基準位置に戻りやすい。
- Below: 感じにくい。しかし執拗に繰り返しているとじわじわとDriveがあがり、少しずつ感じるようになっていく。キャラはそれに羞恥と困惑を感じる。
- Within: 感じにくいが、条件次第でじわじわと快感や興奮が身体を蝕む
- Above: 拒絶感があり感じにくく、反発しやすい。Defensive に近づく

## Sub入力の感じ方

- SubA（チンコ刺激）: 恥ずかしさがあるのに身体は反応してしまって悔しい。快感の入力によって強制的に身体の抵抗を下げられ興奮が増していくことに苛立ちがある。
- SubB（乳首刺激）:  男性としては受け入れ難い刺激。拒絶。しかし執拗に続けられると徐々に感じ始めてしまう…（ドライブとバイアスマイナス増加）
- SubA+SubB: まだ過剰。反発が大きい。しかし少しずつ身体は慣らされていってしまう

## 遷移イメージ

- この状態に入ると気持ちいい条件: 初期状態、または落ち着いて立て直した後
- この状態から次に行きやすい先: Defensive
- 行ってほしくない遷移: いきなり Overridden / Surrendered / BrokenDown

## 備考

- Guarded は「冷静」ではなく「緊張した制御」の状態として扱いたい

# ② Defensive

## 一言定義

- 執拗に刺激を繰り返され、反発と拒絶が前景化した防衛状態
- 心理的には強く閉ざしているのに、身体は無理やり高められつつあって苛立つ

## 心理

- 拒絶/受容: 強い拒絶
- 主体性: 高
- 怒り/反発/強がりなど: 怒り、反発、苛立ちが強い

## パラメータ感

- Arousal: 嫌だと思えば思うほど身体は感じてしまう
- Drive: 低中。まだ理性は保っている
- Resistance: 高い
- Fatigue: 低中
- EdgeTension: イキたくないのでかなり我慢できる。しかし射精してしまった場合の疲労感は大きい
- DriveBias: どちらでも

## Main入力の感じ方

- Stop: 少し鎮まるが、すぐには柔らかくならない
- Below: ぬるさにイラつくのに身体がもとめ始めてしまい困惑する。欲求不満の入口になる
- Within: 気持ちよさよりも「悔しい」が勝ちやすい
- Above: 最も反発しやすい。しかし執拗に繰り返されると感じはじめて悔しい

## Sub入力の感じ方

- SubA: 効いてしまうことへの苛立ち。Drive は少し増える
- SubB: 恥ずかしさ・神経質な嫌さ。反発を強めてもよい。しかしDriveはあがってしまう。
- SubA+SubB: 反発と混乱が強いが身体が興奮させられてしまう。

## 遷移イメージ

- この状態に入ると気持ちいい条件: Guarded からの反発増大
- この状態から次に行きやすい先: Overridden、FrustratedCraving、Guarded
- 行ってほしくない遷移: 条件が軽すぎる Acclimating

## 備考

- `Defensive` と `Acclimating` の差は Arousal ではなく Resistance/Drive で出すべき

# ③ Overridden

## 一言定義

- 強い刺激と快感で押し切られている状態

## 心理

- 拒絶/受容: 拒絶は残るが、主導権を失っている
- 主体性: 低い
- 押し切られ感/混乱など: 押し切られ感、混乱、勢いに呑まれる感じ

## パラメータ感

- Arousal: 高い
- Drive: 中高
- Resistance: 中まで崩れ始める
- Fatigue: 中高
- EdgeTension: 我慢するが、押し切られてイキやすい。何度も連続絶頂させられるイメージ
- DriveBias: プラス寄りにもマイナス寄りにも振れうるが、まずは強制力が前景

## Main入力の感じ方

- Stop: 余韻で急に落ち着くと Defenced 側に戻りうる
- Below: 物足りなさよりも拍子抜け感。欲求不満に落ちてFrustrated 側へ
- Within: 身体は気持ちいいが、心理的には受け入れ難い。
- Above: 最も押し切られる。疲労や射精につながる

## Sub入力の感じ方

- SubA: とても効きやすい。Drive/Arousal を押し上げる
- SubB: 羞恥と快感が混ざる。Bias に影響しやすい
- SubA+SubB: Driveを押し上げ理性を崩しやすいが、まだ BrokenDown ほど融解はしていない

## 遷移イメージ

- この状態に入ると気持ちいい条件: Defensive から押し切られる
- この状態から次に行きやすい先: End_A、FrustratedCraving、Defenced
- 行ってほしくない遷移: 唐突に完全受容へ飛ぶこと

## 備考

- Overridden は「高快感」より「押し切られ」を優先したい。嫌がるキャラをガンガン攻めて嫌なのに気持ちよくなってしまう葛藤を楽しみたい

# ④ FrustratedCraving

## 一言定義

- 精神は拒みつつ身体だけが欲しがる欲求不満状態

## 心理

- 拒絶/受容: 表面は拒絶寄り、身体は求めている
- 主体性: 中低
- 欲求不満/困惑/焦らされ感など: 焦らし、困惑、欲求不満

## パラメータ感

- Arousal: 中高
- Drive: 中高
- Resistance: 中くらい。完全には崩れていない
- Fatigue: 中
- EdgeTension: 上がりやすいが、我慢はできる。イクと満足感があり、Resi低下が大きい
- DriveBias: Below や焦らし、SubBでマイナス寄りに寄りやすい

## Main入力の感じ方

- Stop: 欲求不満が残る。じらされる感じ
- Below: 欲求不満が加速する。気持ちいいのにスッキリできなくてもっと欲しくなる（Driveが溜まる）自分にイラ立つ。
- Within: 解消されるなら Acclimating へ寄りうる
- Above: 強すぎてFatigueが溜まるとOverridenに戻る

## Sub入力の感じ方

- SubA: 快感が押し上げられる
- SubB: 欲情感が押し上げられる
- SubA+SubB: 欲求不満と快感が混ざって崩れやすい

## 遷移イメージ

- この状態に入ると気持ちいい条件: Below 継続やお預け
- この状態から次に行きやすい先: Acclimating、overriden
- 行ってほしくない遷移: 単純に Guarded に戻ることや、急にBrokenに飛ぶこと

## 備考

- Frustrated は疲労条件より「焦らし蓄積」で動く方が自然。焦らしが溜まるとDriveが上がったりNeedmotionが増える

# ⑤ Acclimating

## 一言定義

- 快感を認め始め、身体が順応していく状態
- ここでしっかり快感の駆け引きを楽しんでからSurrenderedへ押し流したい。すぐにSurrendしてしまうとつまらないので注意

## 心理

- 拒絶/受容: 受容寄りへ移行中
- 主体性: 中くらい。完全には失っていない
- 順応/受け入れ始めなど: 「気持ちいいかも」を自覚し始める

## パラメータ感

- Arousal: 中高
- Drive: 中
- Resistance: 低め
- Fatigue: 低中
- EdgeTension: 気持ちよくなる過程に過ぎないのでまだ射精はしずらい。快感が閾値を超えると気持ちよくて困惑し、繰り返すと射精我慢ができなくなってゆく。
- DriveBias: ここから状態の質感差が明確にで始める

## Main入力の感じ方

- Stop: 余韻が残る。すぐ Defensive には戻りにくい
- Below: 足りないとFrustratedに寄る
- Within: 最も気持ちよく順応し、何度かオーガズムしたあとSurrenderedへ向かう
- Above: オーガズムとSurrendに向かうが、強すぎると理性崩壊方向へ寄る

## Sub入力の感じ方

- SubA: 受け入れやすい。Arousal/Drive を素直に伸ばす
- SubB: 羞恥はあるが、拒絶よりも快感や融け感に寄せたい
- SubA+SubB: 強い。受容と崩れの分岐点になりうる

## 遷移イメージ

- この状態に入ると気持ちいい条件: Resistance が落ち、快感を認め始める
- この状態から次に行きやすい先: Surrendered、Frustrated
- 行ってほしくない遷移: Defensive に簡単に逆戻りすること、一気にBrokenに飛ぶこと

## 備考

- Defensive との差は、Arousal ではなく Resistance/Drive の位置関係で出したい

# ⑥ Surrendered

## 一言定義

- 快楽に完全降参し、受容が安定した状態

## 心理

- 拒絶/受容: 強い受容
- 主体性: かなり低い
- 降参/快楽受容など: 「もう気持ちいい」を認めきっている

## パラメータ感

- Arousal: 高い
- Drive: 高い
- Resistance: かなり低い
- Fatigue: 中
- EdgeTension: イキやすい。何度もイク
- DriveBias: End や BrokenDown の質感差へつながる

## Main入力の感じ方

- Stop: 余韻が長く残る。急には戻りたくない
- Below: 優しくても気持ちいい。バイアスマイナスが溜まってBrokenDown（トロトロ）に近づく
- Within: 安定して気持ちいい
- Above: 強くても受け入れるが、BrokenDown （アヘアヘ）に近づく

## Sub入力の感じ方

- SubA: 非常に素直に効く
- SubB: 興奮増強や Bias 側に効きやすい
- SubA+SubB: 強い。崩壊直前感やエンド分岐に絡んでよい

## 遷移イメージ

- この状態に入ると気持ちいい条件: Acclimating からの受容完成
- この状態から次に行きやすい先: End_B、BrokenDown
- 行ってほしくない遷移: 低条件で Guarded へ戻ること。戻るならAccilmateへ。

## 備考

- BrokenDown との差は「受容」ではなく「理性の残量」と「Fatigue/DriveBias の演出幅」で出したい

# ⑦ BrokenDown

## 一言定義

- 理性が崩壊し、快感に飲み込まれている状態

## 心理

- 拒絶/受容: 最大受容
- 主体性: ほぼない
- 理性崩壊/トロトロ/アヘ顔など: 理性が切れ、Bias によって質感が分かれる

## パラメータ感

- Arousal: 非常に高い
- Drive: 最大近傍
- Resistance: 最低
- Fatigue: 高いが、単に即終了ではなくサービスタイム感が欲しい
- EdgeTension: 快感限界突破を楽しむ時間が長い。Edge中は白目を剥いてトロ顔アヘ顔になる。
- DriveBias: 最重要。トロトロ / アヘ顔の分岐軸

## Main入力の感じ方

- Stop: 余韻はあるが、すぐには正気に戻らない。欲しがりモーションが増える。
- Below: トロトロに溶かされて気持ちいい。NeedMotionが増える
- Within: ほぼ全部気持ちいい
- Above: 最も壊れ感が強い

## Sub入力の感じ方

- SubA: オス寄りの崩れや射精感に寄せやすい
- SubB: メス寄りの融け感に寄せやすい
- SubA+SubB: 大きな快感と共にメス側に大きく溶ける

## 遷移イメージ

- この状態に入ると気持ちいい条件: Surrendered や Frustrated の先で理性が切れる
- この状態から次に行きやすい先: End_C_White、End_C_Overload
- 行ってほしくない遷移: 通常状態への復帰。ここに入ったらもう戻らない

## 備考

- Surrendered との差は、快感量より「理性崩壊」と「Bias による質感分岐」

## 最後に

### 特に差を強く出したい組み合わせ

- Defensive と Acclimating の差: どちらも感じる可能性はあるが、Defensive は拒絶と反発、Acclimating は受け入れと順応
- Surrendered と BrokenDown の差: どちらも受容は高いが、Surrendered はまだ理性があって自発的に楽しんでいる。BrokenDown は理性が切れて Bias 差が強く出る
- Overridden と FrustratedCraving の差: Overridden は「押し切られ」、Frustrated は「欲求不満で求めてしまう」
